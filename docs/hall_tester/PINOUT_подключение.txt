ПИНАУТ ПОДКЛЮЧЕНИЯ ДЛЯ ТЕСТИРОВАНИЯ ДАТЧИКОВ ХОЛЛА AH3134
================================================================

ARDUINO NANO (ATmega328P) - ПОДКЛЮЧЕНИЕ КОМПОНЕНТОВ
====================================================

ПИТАНИЕ:
---------
VIN (пин 30)     -> +5V от USB или внешнего источника питания
GND (пин 29)     -> Общий провод (земля)
5V (пин 27)      -> +5V для питания датчиков Холла

ДАТЧИК ХОЛЛА 1 (AH3134):
-------------------------
VCC (вывод 1)    -> +5V (пин 27 Arduino)
GND (вывод 2)    -> GND (пин 29 Arduino)
OUT (вывод 3)    -> D2 (пин 2 Arduino) - цифровой вход

ДАТЧИК ХОЛЛА 2 (AH3134):
-------------------------
VCC (вывод 1)    -> +5V (пин 27 Arduino)
GND (вывод 2)    -> GND (пин 29 Arduino)
OUT (вывод 3)    -> D3 (пин 3 Arduino) - цифровой вход

ВСТРОЕННЫЕ СВЕТОДИОДЫ ARDUINO NANO:
====================================
L (пин 13)       -> Индикация датчика Холла 1
RX (пин 0)       -> Тестовая индикация при запуске (3 моргания)
TX (пин 1)       -> Индикация датчика Холла 2

ПРИМЕЧАНИЯ ПО ПОДКЛЮЧЕНИЮ:
==========================
1. Датчики Холла AH3134 имеют 3 вывода в корпусе TO-92UA:
   - Вывод 1 (VCC): питание +5V
   - Вывод 2 (GND): общий провод
   - Вывод 3 (OUT): выходной сигнал

2. Датчики работают по принципу "активный низкий" - при поднесении
   магнита выход переходит в состояние LOW (0V)

3. Встроенные светодиоды загораются при обнаружении магнитного поля:
   - L (пин 13) - при обнаружении магнита датчиком 1 (флаг HIGH)
   - RX (пин 0) - 3 моргания при запуске программы
   - TX (пин 1) - при обнаружении магнита датчиком 2 (флаг HIGH)

4. Для стабильной работы используются подтягивающие резисторы
   (встроены в Arduino)

5. НЕ ТРЕБУЮТСЯ внешние светодиоды и резисторы!

6. Используются флаги состояния:
   - sensor1Active = true/false для датчика 1
   - sensor2Active = true/false для датчика 2

ТЕСТИРОВАНИЕ:
==============
1. Загрузите программу hall_sensor_test.ino в Arduino Nano
2. При запуске светодиод RX моргнет 3 раза
3. Откройте монитор порта (скорость 9600 бод)
4. Поднесите магнит к датчику 1 - загорится светодиод L (флаг HIGH)
5. Поднесите магнит к датчику 2 - загорится светодиод TX (флаг HIGH)
6. Уберите магнит - соответствующий светодиод погаснет (флаг LOW)

ДИАГНОСТИКА:
=============
- Если светодиоды не загораются: проверьте подключение датчиков
- Если датчики не реагируют: проверьте питание и подключение
- Если сигнал нестабильный: убедитесь в надежности контактов
- Если RX не моргает при запуске: проверьте загрузку программы
